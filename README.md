# Three.js 3D Graphics Template

A minimal starter template for 3D graphics with [Three.js](https://threejs.org/), [Vite](https://vitejs.dev/), and TypeScript.

This template is designed to be used with [Dyad](https://dyad.sh) as a starting point for AI-assisted 3D graphics projects.

## Tech Stack

- **Three.js 0.172** — popular WebGL 3D library
- **Vite 6** — fast dev server and bundler
- **TypeScript 5.7**

## Getting Started

```bash
npm install
npm run dev
```

Then open the local URL printed in the terminal.

## Scripts

- `npm run dev` — start the Vite dev server with HMR
- `npm run build` — type-check and build for production
- `npm run build:dev` — build without type-checking (faster iteration)
- `npm run preview` — preview the production build locally

## Project Structure

```
src/
  main.ts        # entry point, renderer bootstrap
  scenes/        # Three.js scenes
```
